# Image Prompts

## Shared Series Identity

Use a consistent Magick AI plugin-family visual language across future assets:

- Modern open-source developer tooling.
- Clean geometric shapes.
- Deep charcoal or clean off-white base.
- WordPress-like blue accents.
- Green validation accents.
- A small magenta/magic accent used sparingly as the Npcink family signature.
- No official WordPress logo, MCP logo, AI company logos, or trademarked marks.

## Icon Prompt

```text
Create a square WordPress.org plugin icon for "Npcink Governance Core", one plugin in the Npcink plugin family.

This specific plugin represents the local WordPress governance layer for AI-assisted operations: proposal intake, approval boundaries, commit preflight, scoped app keys, and audit logs. It is not the ability content package, not an OpenClaw adapter, not a cloud connector, and not a model client.

Visual metaphor: a precise governance shield surrounding a proposal document or operation card, with approval gates, preflight check lines, and a compact audit trail. The image should feel like a trusted local control-plane component inside a larger plugin suite.

Series identity: modern open-source developer tooling, clean geometric shapes, consistent Npcink family style, restrained technical design.

Color direction: deep charcoal or ink base, WordPress-like blue accents, green approval/check accents, small magenta/magic accent used sparingly as the Npcink family signature.

Style: flat vector, crisp, high contrast, professional, readable at 128x128 and 256x256.

Avoid: robot faces, chat bubbles, cloud product imagery, ability-node content-pack visuals, network adapter plugs, official WordPress logo, MCP logo, AI company logos, tiny unreadable text, fantasy magic wand imagery.
```

## Banner Prompt

```text
Create a WordPress.org plugin banner for "Npcink Governance Core", size 1544x500, with a layout that also works when scaled down to 772x250.

This plugin is part of the Npcink plugin family. Its role is the local WordPress governance layer: ability intake, proposal records, approval and rejection decisions, commit preflight, scoped app keys, and audit logs for AI-assisted WordPress operations.

It complements the other Magick AI plugins:
- Npcink Abilities Toolkit handles ability definitions and ability callbacks.
- trusted adapter handles OpenClaw channel adaptation and calls Core plus the Abilities API.
- cloud connector links to cloud services.
This banner should clearly represent Core governance, not Abilities, Adapter, or Cloud.

Visual concept: a governance console represented abstractly as proposal cards moving through approval gates, preflight validation, and audit timeline evidence. Show a local WordPress control-plane boundary, not cloud execution or chatbot UI.

Text: "Npcink Governance Core"
Optional subtitle: "Governance, approval, preflight, and audit for WordPress AI operations"

Tone: trusted control plane, secure review, operational accountability, calm and professional developer infrastructure.

Series identity: use a visual language that can extend to Abilities, Adapter, and Cloud Addon: same geometry, same base palette, different symbolic focus.

Color direction: deep charcoal or clean off-white base, WordPress-like blue, green approval/check accents, small magenta/magic accent used sparingly.

Composition: strong readable title area, abstract governance flow on the opposite side, generous margins, no critical details near edges, balanced enough for LTR and RTL contexts.

Avoid: robots, chatbot UI, cloud dashboards, ability catalog nodes as the main metaphor, network bridges, official WordPress logo, MCP logo, AI company logos, fantasy magic effects, marketing hype, small unreadable code text.
```
