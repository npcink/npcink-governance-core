# WordPress.org Stable Readme Translation Drafts

These files are source drafts for WordPress.org `Stable Readme` translation
submissions. They are not loaded by the plugin at runtime and are intentionally
kept in `sj/`, outside the release package.

Submit these translations through:

https://translate.wordpress.org/projects/wp-plugins/npcink-governance-core/

## Submission Notes

1. Open the plugin project on translate.wordpress.org.
2. Pick the target locale.
3. Translate the `Stable Readme` strings first, because those power the public
   plugin directory description, installation, FAQ, and changelog sections.
4. Submit matching runtime strings under `Stable` after the readme strings.
5. Ask the locale team or a Project Translation Editor to review and approve
   the submitted strings.

Keep brand, slug, REST namespace, and technical identifiers stable:

- `Npcink Governance Core`
- `Npcink AI`
- `npcink-governance-core`
- `/wp-json/npcink-governance-core/v1/`
- `WordPress Abilities API`
- `commit preflight`
- `app key`

Do not localize the slug or REST namespace.
